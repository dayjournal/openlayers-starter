# openlayers-starter

![README02](img/README02.png)

Start OpenLayers easily.  
- [OpenLayers v6.15.1](https://openlayers.org)  
- [TypeScript v4.7.4](https://www.typescriptlang.org)  
- [Vite v3.0.4](https://vitejs.dev)  
- node v18.1.0
- npm v8.8.0

<br>

## Usage

![README03](img/README03.png)

<br>

Install package
```bash
npm install
```

<br>

build
```bash
npm run build
```

<br>

dev
```bash
npm run dev
```
