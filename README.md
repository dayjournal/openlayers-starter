# openlayers-starter

![README02](img/README02.png)

![README05](img/README05.png)

Start OpenLayers easily.  
- [OpenLayers v6.13.0](https://openlayers.org) 
- [webpack v5.49.0](https://webpack.js.org)  
- node v16.10.0
- npm v7.24.0

<br>

## Usage

![README04](img/README04.png)

<br>

Install package
```bash
npm install
```

<br>

build
```bash
npm run build
```

<br>

dev
```bash
npm run dev
```
